Sumador

Detalles:

Sumador de base unaria. Esta máquina determina el resultado de una suma entre N números en base unaria.

Uso:

Término+Término=

Ejemplos:

111+11=
1+1+1+1+1=

Restador

Detalles:

Restador de base unaria. Esta máquina determina el resultado de la resta entre DOS números en base unaria.

Uso:

'Término a'-'Término b'=
a != b ; a >= 1 ; b >= 1

Ejemplos:

111-11=
11-111=

Duplicador

Detalles:

Duplicador de cadenas. Esta máquina duplica una cadena de símbolos a, b o c de forma idéntica.

Ejemplos:

aaabbbccc
ccaaaacab
cbacbacba

Palíndromos

Detalles:

Verificador de palíndromo. Esta máquina verifica si una cadena es un palíndromo. Si la cinta queda vacía, la cadena es un palíndromo.

Ejemplos:

aaabbbaaa
ccaaaacc
cbacbcabc

Multiplicador

Detalles:

Multiplicador de base unaria. Esta máquina determina el resultado de la multiplicación entre DOS números en base unaria.

Uso:

'Término a'*'Término b'=
a >= 1 ; b >= 1

Ejemplos:

111*11=
11*111=
